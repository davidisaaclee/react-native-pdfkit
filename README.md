# react-native-pdfkit

React Native wrapper around some Apple PDFKit APIs

## Installation


```sh
npm install react-native-pdfkit
```


## Usage


```js
import { PdfkitView } from "react-native-pdfkit";

// ...

<PdfkitView color="tomato" />
```
